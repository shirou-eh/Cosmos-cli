──┤ НАЧАЛО РАБОТЫ ├──

VoidCorp Terminal — симулятор космической программы прямо в терминале. Собирай ракеты из деталей, запускай их, выполняй контракты спонсоров, исследуй планеты и открывай новые технологии.

terminal УСТАНОВКА 
$ curl -fsSL https://cosmos-cli.xyz/install | bash
[ OK ] Python 3.9+
[ OK ] Установка завершена
$ cosmos
Требования: Python 3.9+, терминал с True Color (Windows Terminal / iTerm2 / GNOME Terminal)
──┤ СБОРКА РАКЕТ (VAB) ├──

В Сборочном Цехе (VAB) ты собираешь ракету из деталей. Нажми [V] в главном меню.

Детали

КатегорияКлавишаОписание
Командный модуль[1]Капсула экипажа. Без неё ракета не взлетит.
Топливные баки[2]Хранят топливо. Разной ёмкости.
Двигатели[3]Создают тягу. Есть атмосферные и вакуумные.
Утилиты[4]Парашюты, бур, солнечные панели, и многое другое.

Боковые ускорители

Используй клавишу [A] для переключения стороны установки: ЦЕНТР → ЛЕВЫЙ БОК → ПРАВЫЙ БОК. Так можно строить ракеты с боковыми ускорителями (например, как Союз).

Ступени

Разделитель ([5]) создаёт новую ступень. Детали до разделителя — верхняя ступень, после разделителя — нижняя (она запускается первой и отбрасывается).

Система COMBO

Устанавливай детали быстро (в течение 2 секунд) — получишь скидку до 50% на всю ракету.

Советы по полёту

Gravity Turn: Начни наклон на 10 км. Плавно уменьшай тангаж по мере подъёма. На высоте 30–40 км угол должен быть ~30°. На 70–100 км — почти горизонтально.
Орбита: Целевая скорость для орбиты Терры — ~2800 м/с на высоте 100 км. Следи за апогеем и перигеем в бортовом компьютере.
Посадка: Включи парашюты при скорости снижения и высоте < 5000 м. Используй двигатели для мягкого касания на безатмосферных телах.
Perfect Staging: Отстреливай ступень при < 5% топлива — получишь бонус +50% к тяге следующей ступени.

В игре используется симуляция орбитальной механики с RK4-интегратором.

Дельта-V

В VAB показывается расчётная Δv ракеты с учётом ступеней. Формула: Δv = Isp × g₀ × ln(m₀/m₁)

TWR

Отношение тяги к весу. TWR < 1.0 — ракета не взлетит. TWR 1.0–1.5 — взлетит, но медленно. TWR > 1.5 — оптимально.

Расход топлива

Учитывает Isp двигателя и атмосферное давление. В атмосфере двигатели работают менее эффективно.

Атмосфера

Плотность падает с высотой. На границе атмосферы сопротивление минимально. Парашюты работают только в атмосфере.
